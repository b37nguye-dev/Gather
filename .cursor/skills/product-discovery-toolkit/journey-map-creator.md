# Journey Map Creator (Reference)

End-to-end experience maps with pain points and opportunities.

## Process

1. Check personas.md, product.md, research files
2. Define scope: persona, scenario, start/end
3. Gather journey data (interviews, analytics, support)
4. Map 4–7 stages (e.g., Awareness → Consideration → Decision → Onboarding → Usage → Advocacy)
5. For each stage: Actions, Thoughts, Emotions, Touchpoints, Pain Points
6. Identify opportunities (prioritize by severity, frequency, feasibility)

## Output Structure

- Context, data sources, confidence
- Journey overview table (Stage, Actions, Emotions, Pain Points)
- Stage details (Actions, Thoughts, Emotions, Touchpoints, Pains, Opportunities)
- Emotional journey visualization
- Moments of truth table
- Priority opportunities
- Connection to roadmap

## Framework

Service design. Include emotions. Focus on moments of truth (make-or-break points).
