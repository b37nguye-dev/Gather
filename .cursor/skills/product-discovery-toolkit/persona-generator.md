# Persona Generator (Reference)

Build behavioral buyer and user personas with Pragmatic Institute use scenarios, JTBD, and buying process.

## Process

1. Check personas.md, product.md, company.md, research files
2. Request research (interviews, surveys, analytics) — flag as assumed if thin
3. Distinguish Buyers (Economic/Functional/Technical) from Users (B2B)
4. Create use scenarios: Persona + Goal + Problem + Frequency
5. Extract jobs-to-be-done
6. Map buying process (Awareness → Evaluation → Decision → Implementation)
7. Define anti-personas (who you're NOT building for)

## Output Structure

- Overview, use scenarios table, JTBD, goals, frustrations
- Behaviors (triggers, workflow, frequency)
- Buying process table (if buyer)
- Evidence (Validated vs. Assumed)
- Product implications table
- Anti-persona with "Why Not" and "Risk if we target them"

## Framework

Pragmatic Institute personas + Jobs-to-be-Done. Focus on behavior, not demographics.
