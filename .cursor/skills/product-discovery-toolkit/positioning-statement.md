# Positioning Statement Generator (Reference)

April Dunford's Obviously Awesome framework.

## Process

1. Check product.md, personas.md, competitors.md, company.md
2. Identify competitive alternatives (direct, indirect, status quo)
3. Define unique attributes (from product.md)
4. Map attributes to customer value (from personas.md)
5. Identify best-fit customers
6. Determine market category (existing, subcategory, new)
7. Synthesize positioning statement

## Output Structure

- Positioning canvas table (Alternatives, Attributes, Value, Best-fit, Category)
- Statement: **For** [customers] **who** [need], **[Product]** is a [category] **that** [benefit]. **Unlike** [alternatives], **we** [differentiator].
- Tagline options (Benefit-led, Category-led, Differentiator-led)
- Elevator pitch (30 sec)
- Competitive frame
- Messaging by persona table

## Framework

Positioning = strategic context. Start with alternatives, not features. One positioning, multiple messages.
