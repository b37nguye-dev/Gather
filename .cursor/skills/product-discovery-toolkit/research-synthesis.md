# Research Synthesis Engine (Reference)

Combine multiple research sources into themes and actionable recommendations.

## Process

1. Check personas.md, product.md, prior research files
2. Gather 3+ sources (interviews, surveys, support, sales notes, app reviews)
3. Atomic breakdown: one insight per observation, source, evidence strength
4. Theme clustering: patterns across sources, evidence strength
5. Pattern analysis: Convergence, Contradictions, Outliers, Gaps
6. Connect to existing knowledge (personas, product, roadmap)

## Output Structure

- Sources analyzed table
- Executive summary
- Theme analysis (Theme, Evidence Strength, Key Insights, Quote, Connection)
- Patterns: Convergence, Contradictions, Outliers, Gaps
- Impact on personas and roadmap tables
- Prioritized opportunities (Evidence, Impact, Confidence)
- Recommended next steps
- Suggested updates to context files

## Framework

Atomic Research + thematic analysis. More sources = higher confidence. Contradictions reveal segments.
