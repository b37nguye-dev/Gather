# Roadmap Builder (Reference)

Now/Next/Later roadmap linked to business objectives and product outcomes.

## Process

1. Check for strategic-roadmap-review output; use if exists
2. Read product.md, company.md, personas.md, competitors.md
3. Map Business Objectives → Product Outcomes → Initiatives
4. Organize into Now (immediate), Next (after Now), Later (deferred)
5. Identify constraints, dependencies, risks
6. Define success metrics (pull from product.md; use PLACEHOLDER if no baseline)

## Output Structure

- Context (what was pulled from files)
- Business objectives table (Objective, Source, Key Metric)
- Leading product outcomes table (Outcome, Drives Objective, Current, Target)
- Now/Next/Later tables (Initiative, Drives Outcome, Owner, Dependencies)
- Dependencies & Risks table
- Not This Period (parking lot)
- Assumptions to validate

## Framework

Objective → Outcome → Initiative chain. Now/Next/Later (Janna Bastow/ProdPad).
